# Introduction
IF.NET is a free and open-source [.NET] library for creating interactive fiction. 

## What Is Interactive Fiction
Interactive Fiction (or simply IF) refers to fiction in which players type commands to interact with a virtual world. For a good example, see the [Zork](https://en.wikipedia.org/wiki/Zork) series.

## Why IF.NET?
* IF.NET is free and open-source. Its permissive MIT license allows you to use it in any way you like, provided you follows [its terms](https://git.gordinator.org/Gordinator/IFDotNet/src/branch/master/LICENSE).
* IF.NET has a powerful .NET API that allows experienced (or beginner) .NET developers to create interactive fiction with unlimited flexibility.
* The engine is highly extensible, allowing you to add custom functionality.
